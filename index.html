    NumCry

    Like NumPy, but slower!

    Useful methods for numerical computing in crystal-lang.

    Installation

    1. Add the dependency to your shard.yml:

    `yaml dependencies:

     numcry:
   github: aca-labs/numcry

    `

    1. Run shards install

    Usage

    require "numcry"
